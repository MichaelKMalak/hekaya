# أمثلة — Examples

## ١. محادثة كاملة: كتابة مشهد من الصفر (Full Conversation)

**User:** عايز أكتب مشهد في قهوة بلدي، سمير بيقابل صاحبه حسن

**Assistant:**

```hekaya
داخلي - قهوة بلدي - نهار

قهوة قديمة في وسط البلد. التليفزيون شغال على ماتش محدش بيتفرج عليه. سمير (٣٥ سنة) قاعد في الركنة بتاعته — القعدة اللي كل يوم بيقعد فيها.

@سمير
(بيقلّب في الشاي من غير ما يشرب)

حسن بيدخل. ماسك أكياس. لابس بدلة مكرمشة.

@حسن
يا سمير! أنا من الصبح بدور عليك.

@سمير
(من غير ما يرفع راسه)
وأنا من الصبح هنا.

@حسن
(بيقعد قصاده)
طب ما كنت رديت على التليفون.

سمير بيبص لحسن أول مرة. بيشوف البدلة والأكياس.

@سمير
رايح فرح ولا جاي من فرح؟

@حسن
(بيتنهد)
جاي من المحكمة.

(سكوت طويل)

@سمير
(بيلوّح للجرسون)
شايين يا معلم.
```

### إيه اللي حصل هنا syntax-wise:

- **`داخلي - قهوة بلدي - نهار`** — scene heading بالعربي، dash separator
- **`@سمير`** — أول استخدام، بيسجل الاسم في Character Registry
- **`@حسن`** — أول استخدام لحسن، بيتسجل برضو
- **`سمير`** بعد كده (سطر ١٧) — من غير `@` عشان اتسجل فعلاً
- **`(بيقلّب في الشاي)`** — parenthetical على سطر لوحده
- **Action** — أي فقرة عادية = وصف/action
- مفيش transitions هنا عشان المشهد مكمّل — لو عايز تقطع بتكتب `قطع` أو `- قطع -`

## ٢. Input/Output: عناصر منفردة (Element Pairs)

### Title Page

**Input:**

```
العنوان: آخر أيام الصيف
سيناريو: سمير عبدالحميد
مسودة: المسودة الأولى
تاريخ: ٢٠٢٦/٣/٢٩
تواصل: samir@example.com
```

**Output:** Title page عادية. كل سطر = key: value. الـ blank line بعد آخر key بيفصل الـ title page عن الـ body.

### Title Page بأكتر من مؤلف

**Input:**

```
العنوان: حكاية شارع
كتابة:
    سمير عبدالحميد
    ونادية حسن
```

**Output:** Multi-line value — كل سطر بعد الـ key بيكون indented بـ spaces أو tabs.

### Scene Heading

**Input:**

```
خارجي/داخلي - عربية متحركة - غروب
```

**Output:** Scene heading — `خارجي/داخلي` = EXT/INT, `عربية متحركة` = location, `غروب` = time of day.

### Scene Heading برقم مشهد

**Input:**

```
داخلي - مكتب المدير - نهار #٣#
```

**Output:** Scene heading مع scene number `٣`.

### Forced Scene Heading

**Input:**

```
.في مكان ما
```

**Output:** أي سطر بيبدأ بنقطة `.` بيتعامل كـ scene heading حتى لو مفيش keyword.

### Character + Dialogue

**Input:**

```
@ليلى
مش عايزة أتكلم في الموضوع ده تاني.
```

**Output:** Character cue (`ليلى`) + dialogue. `@` بتسجل الاسم.

### Character بعد التسجيل

**Input:**

```
ليلى
طيب.
```

**Output:** بعد ما `@ليلى` اتكتبت مرة، `ليلى` لوحدها على سطر + حوار بعدها = character element تلقائياً.

### Character Extension

**Input:**

```
@سمير (صوت من خارج المشهد)
أنا مش هنا بس بسمعكم.
```

**Output:** Character + V.O. extension. بدائل: `(ص.خ)`, `(خارج الشاشة)`, `(خ.ش)`, `(خارج الكادر)`, `(تابع)`.

### Dual Dialogue

**Input:**

```
@سمير
أيوه!

@نادية ^
لأ!
```

**Output:** الـ `^` بعد اسم الشخصية التانية = dual dialogue (حوار متوازي جنب بعض).

### Parenthetical

**Input:**

```
@حسن
(بيبص للأرض)
مش عارف أقولك.
(سكوت)
بس لازم تعرف.
```

**Output:** Parentheticals بين أقواس على سطر لوحدهم جوّه block الحوار. ممكن أكتر من واحد.

### Transition

**Input:**

```
- اختفاء تدريجي -
```

**Output:** Transition — keyword عربي بين dashes. بدون dashes بيشتغل برضو:

```
قطع
```

### Forced Transition

**Input:**

```
>اللي حصل قبل كده
```

**Output:** أي سطر بيبدأ بـ `>` (من غير `<` في الآخر) = forced transition.

### Centered Text

**Input:**

```
>النهاية<
```

**Output:** `>` في الأول و `<` في الآخر = نص في النص.

### Emphasis

**Input:**

```
سمير بيمسك **الجواب** وإيده *بترتعش*.
```

**Output:**

- `**الجواب**` → bold
- `*بترتعش*` → italic
- `***text***` → bold italic
- `_text_` → underline

### Page Break

**Input:**

```
===
```

**Output:** ٣ أو أكتر equals signs = page break.

### Section (مش بيتطبع)

**Input:**

```
# الفصل الأول
## المواجهة
```

**Output:** Sections بـ `#` — للتنظيم بس، مش بتظهر في الـ output.

### Synopsis (مش بيتطبع)

**Input:**

```
= سمير بيقابل حسن في القهوة وبيعرف إنه اتطلق.
```

**Output:** Synopsis — ملخص المشهد للكاتب، مش بيظهر في الـ output.

### Note

**Input:**

```
سمير بيمشي [[لازم نصور في وسط البلد مش في الاستوديو]] في الشارع.
```

**Output:** Inline note بين `[[` و `]]` — بيتشال من الـ output الافتراضي. استخدم `--include-notes` في الـ CLI عشان يظهر.

### Boneyard/Comment

**Input:**

```
/* المشهد ده محتاج إعادة كتابة كاملة
   الحوار ضعيف والإيقاع بطيء */
```

**Output:** Comment — دايماً بيتشال من الـ output.

### Lyrics

**Input:**

```
~يا ليل يا عين
~يا ليل يا عين
```

**Output:** كل سطر بيبدأ بـ `~` = lyrics.

## ٣. Before/After: تحويل من Fountain لـ Hekaya

### Before — Fountain (English conventions)

```fountain
Title: The Last Summer
Author: Samir Abdulhamid
Draft date: March 2026

INT. CAFÉ - DAY

An old café downtown. SAMIR (35) sits alone in his usual corner.

SAMIR
(stirring his tea absently)
Same view. Every day.

HASSAN enters, carrying bags, wearing a wrinkled suit.

HASSAN
Samir! I've been looking for you all morning.

CUT TO:

EXT. STREET - DAY
```

### After — Hekaya (Arabic-first)

```hekaya
العنوان: آخر أيام الصيف
سيناريو: سمير عبدالحميد
تاريخ المسودة: مارس ٢٠٢٦

داخلي - قهوة بلدي - نهار

قهوة قديمة في وسط البلد. سمير (٣٥ سنة) قاعد لوحده في ركنه.

@سمير
(بيقلّب في الشاي)
نفس المنظر. كل يوم.

حسن بيدخل. ماسك أكياس. لابس بدلة مكرمشة.

@حسن
يا سمير! أنا من الصبح بدور عليك.

قطع

خارجي - شارع - نهار
```

### إيه اللي اتغير

| Fountain            | Hekaya                     | ليه                                       |
| ------------------- | -------------------------- | ----------------------------------------- |
| `Title:`            | `العنوان:`                 | Arabic title keys                         |
| `SAMIR` (uppercase) | `@سمير`                    | العربي مفيهوش uppercase — `@` بتسجل الاسم |
| `INT. CAFÉ - DAY`   | `داخلي - قهوة بلدي - نهار` | Arabic keywords + dash separator          |
| `CUT TO:`           | `قطع`                      | Arabic transition keyword                 |
| `EXT.`              | `خارجي`                    | Arabic scene heading keyword              |

## ٤. أخطاء شائعة وإزاي تصلحها (Common Mistakes)

### غلطة ١: نسيت `@` في أول استخدام

```
سمير
قهوة سادة.
```

**المشكلة:** `سمير` مش مسجل في Character Registry — الـ parser مش هيعرف إنه character.

**الصح:**

```
@سمير
قهوة سادة.
```

### غلطة ٢: مفيش blank line قبل الـ character

```
سمير بيقوم من على الكرسي.
@حسن
فين رايح؟
```

**المشكلة:** الـ parser ممكن يلخبط لو مفيش blank line تفصل الـ action عن الـ character.

**الصح:**

```
سمير بيقوم من على الكرسي.

@حسن
فين رايح؟
```

### غلطة ٣: Title page متوصلة بالـ body

```
العنوان: الفيلم
سيناريو: سمير
داخلي - قهوة - نهار
```

**المشكلة:** مفيش blank line بعد الـ title page — الـ parser مش هيعرف فين الـ title page خلصت.

**الصح:**

```
العنوان: الفيلم
سيناريو: سمير

داخلي - قهوة - نهار
```

### غلطة ٤: Parenthetical على نفس سطر الحوار

```
@سمير
(بهدوء) مش عايز حاجة.
```

**المشكلة:** الـ parenthetical لازم يكون على سطر لوحده.

**الصح:**

```
@سمير
(بهدوء)
مش عايز حاجة.
```

### غلطة ٥: استخدام نقطة بدل dash في scene heading

```
داخلي. قهوة بلدي. نهار
```

**المشكلة:** بيشتغل بس مش الـ recommended style. الـ dash separator أوضح وأقرب للمعيار المصري.

**الأحسن:**

```
داخلي - قهوة بلدي - نهار
```

## ٥. CLI Examples (أوامر عملية)

### Parse مشهد لـ JSON

```bash
$ hekaya parse script.hekaya
```

Output:

```json
[
  {
    "type": "scene_heading",
    "text": "داخلي - قهوة بلدي - نهار",
    "scene_number": null
  },
  {
    "type": "action",
    "text": "قهوة قديمة في وسط البلد."
  },
  {
    "type": "character",
    "name": "سمير",
    "extension": null
  },
  {
    "type": "dialogue",
    "text": "قهوة سادة، لو سمحت."
  }
]
```

### Render لـ HTML

```bash
$ hekaya render script.hekaya -o screenplay.html
```

### Export لـ PDF

```bash
$ hekaya export script.hekaya -o screenplay.pdf
```

### Export لـ PDF مع ملاحظات

```bash
$ hekaya export script.hekaya --include-notes -o screenplay.pdf
```

### Validate

```bash
$ hekaya validate script.hekaya
```

### Convert لـ Fountain

```bash
$ hekaya convert script.hekaya -o script.fountain
```

## ٦. سيناريو كامل (Complete Example)

```hekaya
العنوان: قهوة سادة
سيناريو: نادية حسن
مسودة: المسودة الثانية
تاريخ: ٢٠٢٦/٣/٢٩
تواصل:
    nadia@example.com
    ٠١٠٠٠٠٠٠٠٠٠
ملاحظات:
    فيلم قصير - ٧ دقايق
    مصوّر بالكامل في قهوة بلدي واحدة

# الفصل الأول — التأسيس

= سمير بيقابل نادية بالصدفة في القهوة. أول مرة يشوفها من ٥ سنين.

داخلي - قهوة بلدي - نهار

قهوة قديمة. موزاييك على الحيطان. شمس الضهر داخلة من شباك مكسور. المكان فاضي إلا من عم حسن ورا الكاونتر.

@سمير
(قاعد لوحده، بيلعب بعلبة سجاير فاضية)

@عم حسن
(من ورا الكاونتر)
هتطلب ولا هتقعد تبص؟

@سمير
شاي.

@عم حسن
زي كل يوم.

الباب بيتفتح. نادية (٣٠ سنة) بتدخل. واقفة عند الباب لحظة — بتفتكر المكان.

سمير بيرفع راسه. بيشوفها. بيرجع يبص في الشاي.

@نادية
(للجرسون)
لو سمحت... قهوة سادة.

[[الإيقاع هنا مهم — لازم نحس إن الوقت واقف]]

@سمير
(من غير ما يبصلها)
نادية.

(سكوت)

نادية بتبصله. بتعرفه.

@نادية
...سمير؟

===

# الفصل الثاني — المواجهة

= الحوار بينهم بيبدأ عادي بس بيتحول لمواجهة حقيقية.

داخلي - قهوة بلدي - نهار (بعد شوية)

سمير ونادية قاعدين على نفس الترابيزة. فيه مسافة بينهم.

@نادية
بقالك قد إيه بتيجي هنا؟

@سمير
من يوم ما مشيتي.

@نادية
(بتبص في القهوة)
أنا ما مشيتش. إنت اللي...

@سمير
(بيقاطعها)
عايزة سكر؟

(سكوت طويل)

@نادية
**لا.**

سمير بيبصلها أول مرة من أول المشهد. عينيه فيها كل حاجة مش قادر يقولها.

@سمير
(بهدوء)
طب خلاص.

بيقوم. بيحط فلوس على الترابيزة. بيمشي ناحية الباب.

@نادية
سمير.

سمير واقف عند الباب. ضهره ليها. مش بيلتفت.

@نادية
الشاي بتاعك لسه سخن.

(سكوت)

سمير بيرجع. بيقعد. بيشرب الشاي.

- اختفاء تدريجي -

>النهاية<
```
