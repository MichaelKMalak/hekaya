# مرجع — Reference

## الأُطر النفسية — ملخص سريع

### متى تستخدم كل إطار

| الإطار                    | استخدمه لما                                   | مثال                                                                  |
| ------------------------- | --------------------------------------------- | --------------------------------------------------------------------- |
| **يونج — Archetypes**     | عايز تحدد نوع الشخصية ودوافعها                | البطل اللي عايز يثبت نفسه vs. المتمرد اللي عايز يغيّر النظام          |
| **يونج — Shadow/Persona** | عايز تبني قوس شخصية (القناع → الحقيقة)        | الدكتورة المثالية اللي جواها غضب مكتوم                                |
| **الإنياجرام**            | عايز تعرف إيه اللي بيحصل للشخصية تحت الضغط    | نوع ٣ (المنجز) تحت الضغط بيكدب                                        |
| **ماسلو**                 | عايز تفهم ليه الشخصية بتاخد القرارات دي       | شخصية في مستوى "البقاء" بتاخد قرارات مختلفة عن واحدة في "تحقيق الذات" |
| **إريكسون**               | عايز تحدد السؤال الأساسي بناءً على سن الشخصية | شاب ٢٥ = "أنا مين؟" / راجل ٥٠ = "سبت إيه وراي؟"                       |
| **كوبلر-روس**             | قصة عن خسارة (حد، حلم، هوية، وطن)             | المراحل الخمسة كبنية للفيلم                                           |
| **سارتر/كامو**            | قصة وجودية أو عبثية                           | "سوء النية" كعيب قاتل                                                 |
| **ليفي-شتراوس**           | عايز تحدد التناقض المركزي                     | غني/فقير، نظام/فوضى، تقليد/حداثة                                      |
| **كيشوتنكتسو**            | قصة مبنية على المفاجأة مش الصراع              | فيلم "العدو" فيه مش شخص — فقر، وحدة، بيروقراطية                       |
| **أوزوريس**               | قصة عن منفى، غربة، إعادة بناء هوية بعد صدمة   | البطل مش بيرجع لحياته القديمة — بيبقى حد تاني                         |

## حِرَف السيناريو — Quick Reference

### محركات القصة

| الحرفة                     | القاعدة                                        | التطبيق السريع                         |
| -------------------------- | ---------------------------------------------- | -------------------------------------- |
| **بس/عشان كده**            | بين كل حدث لازم "بس" أو "عشان كده" مش "وبعدين" | لو قولت "وبعدين" — القصة ميتة          |
| **الفجوة (McKee)**         | المسافة بين التوقع والنتيجة                    | كل ما الفجوة أكبر = دراما أقوى         |
| **CDQ**                    | سؤال أيوه/لأ في أول ثانية                      | من غيره المشاهد هيمشي                  |
| **Want vs. Need**          | هدف ظاهر + احتياج داخلي                        | أحسن نهاية: بياخد الـ need مش الـ want |
| **Promise of the Premise** | وعد الفكرة لازم يتحقق                          | لو فكرتك "أعمى في أول ديت" — وريني ده  |

### حِرَف المشهد

| الحرفة                      | القاعدة                                    |
| --------------------------- | ------------------------------------------ |
| **Enter Late, Leave Early** | ادخل والصراع شغال، اخرج أول ما الغرض يتحقق |
| **Scene Turn**              | كل مشهد لازم "يلف" — القيمة العاطفية تتغير |
| **Micro-Reversals**         | جوّه المشهد الكلام بيتأرجح                 |
| **Button**                  | آخر لحظة في المشهد بتدفعك للمشهد اللي بعده |

### حِرَف الحوار

| الحرفة               | القاعدة                                        |
| -------------------- | ---------------------------------------------- |
| **Oblique Dialogue** | بيتكلموا حوالين الموضوع مش فيه                 |
| **محادثة ٢.٥**       | كل واحد فاكر نفس الموضوع بس في وادي تاني       |
| **ديناميكية القوة**  | مين بيسأل؟ مين بيقاطع؟ مين بيغير الموضوع؟      |
| **السكوت كحوار**     | شخصية بتسكت بعد سؤال بتقول أكتر من اللي بتجاوب |

### السرد البصري

| الحرفة          | القاعدة                           |
| --------------- | --------------------------------- |
| **الموتيف**     | صورة بتتكرر ٣ مرات وبتاخد معنى    |
| **البلوكينج**   | مكان الشخصيات في الكادر بيحكي     |
| **أثر كوليشوف** | المونتاج بيخلق المعنى مش الأداء   |
| **اللون**       | تغيير اللون = تغيير القوس الدرامي |

### التلاعب بالجمهور

| الحرفة                    | القاعدة                             |
| ------------------------- | ----------------------------------- |
| **Dramatic Irony**        | المشاهد عارف حاجة الشخصية لأ = توتر |
| **Plant & Payoff**        | كل حاجة بتتعرض لازم تترد            |
| **Red Herring**           | زرعة بتوجه في الاتجاه الغلط         |
| **Emotional Preparation** | جهّز المشاهد — هدوء قبل الضربة      |

## التصنيفات الخمسة للأفلام القصيرة

| الرمز | التصنيف             | الوصف          | أمثلة تطبيق                                              |
| ----- | ------------------- | -------------- | -------------------------------------------------------- |
| 🧨    | Extreme Case        | حالة استثنائية | واحد بيصحى ملوش ذاكرة، ست بتلاقي نفسها في مكان مش مكانها |
| 🏠    | Space Limit         | محصور في مكان  | فيلم كامل في أسانسير، في تاكسي، في أوضة                  |
| ⚖️    | Two Opposites       | نقيضين         | أب وابنه، غني وفقير، مؤمن وملحد                          |
| 🔁    | Switch the Familiar | قلب المألوف    | القصة من وجهة نظر الشرير، أو في زمن تاني                 |
| ⏱️    | Time Limit          | عدّاد وقت      | ١٠ دقايق قبل ما القطر يمشي، ساعة قبل ما الامتحان يبدأ    |

**أقوى الأفلام بتخلط تصنيفين أو أكتر.**
