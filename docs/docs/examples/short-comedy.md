# Short Comedy Example

A satirical short screenplay in Egyptian Arabic with a large ensemble cast, demonstrating multi-character dialogue and escalating action.

## Source (`.hekaya`)

See the full sample file: [`samples/بكرا-السما-ح-تقع.hekaya`](https://github.com/michaelkmalak/hekaya/blob/main/samples/بكرا-السما-ح-تقع.hekaya)

This sample demonstrates:

### Title Page (Arabic keys)

```
العنوان: بكرا السما ح تقع
المؤلف: مايقل
مسودة: المسودة الأولى
تاريخ: ٢٠٢٦/٢/٢٣
تواصل: michael@example.com
```

### Many Characters (7 registered)

- `@مايكل` — the protagonist who read the news online
- `@القهوجي` — the coffee shop worker, first to believe
- `@السواق` — a driver who nearly hits القهوجي
- `@الدكتور` — at the downtown café
- `@المحامي` — agrees because "the internet said so"
- `@الفلاح` — agrees
- `@المكوجي` — agrees
